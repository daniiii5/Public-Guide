# Community's Ultimate Guide by John and Dani
## Attention! This is still a work in progress  
The guide is transitioning to a **community-supported version** for collaboration and updates.  

## FAQ
- **Developer Settings Missing:** Ensure Metaconfig Settings and Overrides are used.
- **Developer Options:** Available on mods with dev options enabled.
- **Flag Issues:** Verify base version or region-specific restrictions.
- **Crashing:** Reset app data if faulty flags cause issues.
- **Support:** Message via Patreon for help.
- **Flags Pre-305.0.0.0.107:** Data mapped for later versions only.
